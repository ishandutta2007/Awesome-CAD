# Awesome-CAD
## Top CAD Softwares Ecosystem

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on 2D/3D Computer-Aided Design & Engineering Tools*  
**Last updated: March 2026**

This repository tracks notable **CAD platforms** and **open-source projects** for professional 2D drafting, 3D modeling, parametric design, BIM, and engineering workflows. These tools are used across architecture, mechanical engineering, product design, manufacturing, and construction.

**Examples** include AutoCAD, ZWCAD, DraftSight, Rayon, LibreCAD, Synaps, FreeCAD, Rhino (Rhinoceros), and BricsCAD (the category leaders). Tools listed here emphasize **precision**, compatibility (DWG/DXF), parametric modeling, and industry-standard features.

**Open-source emphasis**: This section is heavily expanded with every major active project for self-hosting, full customization, no licensing costs, and complete data ownership — ideal for individuals, educators, startups, and organizations seeking freedom from proprietary CAD vendors.

Contributions welcome! Open a PR to add/update entries. Keep descriptions factual and link to official sites.

## Open-Source GitHub Projects

### Dedicated Open-Source CAD Software

- **[FreeCAD](https://github.com/FreeCAD/FreeCAD)**  
  The most powerful open-source parametric 3D CAD modeler. Supports complex assemblies, BIM, FEM analysis, and extensive scripting with Python.

- **[LibreCAD](https://github.com/LibreCAD/LibreCAD)**  
  Mature open-source 2D CAD application with excellent DWG/DXF compatibility and a clean interface.

- **[OpenSCAD](https://github.com/openscad/openscad)**  
  Script-based solid 3D CAD modeler using a programming language approach. Ideal for parametric and algorithmic design.

- **[QCAD](https://github.com/qcad/qcad)**  
  Professional open-source 2D CAD with strong DWG/DXF support and scripting capabilities.

- **[BRL-CAD](https://github.com/BRL-CAD/brlcad)**  
  Powerful cross-platform solid modeling system with ray-tracing and geometry engine used in high-performance applications.

- **[SolveSpace](https://github.com/solvespace/solvespace)**  
  Lightweight, constraint-based parametric 2D/3D CAD modeler with excellent usability.

- **[CadQuery](https://github.com/CadQuery/cadquery)**  
  Python-based scripting library for building parametric 3D models with OpenCascade kernel.

- **[Blender (with CAD plugins)](https://github.com/blender/blender)**  
  Industry-leading open-source 3D creation suite with powerful CAD add-ons (BlenderBIM, MeasureIt, etc.).

- **[IfcOpenShell](https://github.com/IfcOpenShell/IfcOpenShell)**  
  Open-source library and tools for working with Industry Foundation Classes (IFC) BIM models.

- **[PythonOCC](https://github.com/tpaviot/pythonocc-core)**  
  Python wrapper for OpenCascade CAD kernel, enabling powerful scripting and custom CAD applications.

### Additional Strong Open-Source Options

- **[A2plus](https://github.com/kbwbe/A2plus)** — Assembly workbench for FreeCAD.
- **[Assembly4](https://github.com/Zolko-123/FreeCAD_Assembly4)** — Advanced assembly workbench for FreeCAD.
- **[Ondsel](https://github.com/Ondsel-Development/Ondsel)** — Modern FreeCAD-based collaborative CAD platform.
- **[OpenCascade](https://github.com/Open-Cascade-SAS/OCCT)** — Core geometry kernel used by many CAD tools.
- Many community **FreeCAD macros** and **workbenches** for specialized domains (architecture, mechanical, electronics).

**Frameworks for building custom CAD tools**: Combine **FreeCAD**, **CadQuery**, **OpenSCAD**, and **PythonOCC** with scripting languages to create tailored design automation pipelines.
