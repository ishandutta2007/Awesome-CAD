## 🔓 Open-Source GitHub Projects

The best open-source CAD software for self-hosting, customization, and data ownership.

### 🌟 Dedicated Open-Source CAD Software

- **[FreeCAD](https://github.com/FreeCAD/FreeCAD)** 🐧  
  The most powerful open-source parametric 3D CAD modeler. Supports complex assemblies, BIM, FEM analysis, and Python scripting.

- **[LibreCAD](https://github.com/LibreCAD/LibreCAD)** 🗺️  
  Mature 2D CAD application with excellent DWG/DXF compatibility and a clean interface.

- **[OpenSCAD](https://github.com/openscad/openscad)** 📜  
  The programmer's solid 3D CAD modeler. Script-based parametric and algorithmic design.

- **[QCAD](https://github.com/qcad/qcad)** 📐  
  Professional 2D CAD with strong DWG/DXF support and scripting capabilities.

- **[BRL-CAD](https://github.com/BRL-CAD/brlcad)** 🚀  
  Powerful cross-platform solid modeling system with a 30+ year history in military engineering.

- **[SolveSpace](https://github.com/solvespace/solvespace)** 🔗  
  Lightweight, constraint-based parametric 2D/3D CAD modeler with a unique workflow.

- **[CadQuery](https://github.com/CadQuery/cadquery)** 🐍  
  Python library for building parametric 3D models. Think "OpenSCAD but with Python power."

- **[Blender (CAD-focused)](https://github.com/blender/blender)** 🎨  
  Industry-leading 3D suite enhanced with CAD plugins like **BlenderBIM** and **MeasureIt**.

---

### 📦 Additional Open-Source Libraries

- **[IfcOpenShell](https://github.com/IfcOpenShell/IfcOpenShell)** — Essential for working with BIM (IFC) models.
- **[PythonOCC](https://github.com/tpaviot/pythonocc-core)** — Python wrapper for the powerful OpenCascade CAD kernel.
- **[OpenCascade](https://github.com/Open-Cascade-SAS/OCCT)** — The core geometry kernel powering many open-source tools.
